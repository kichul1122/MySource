- 법선벡터(법선 : 수직인 직선)  
2D : 어떤 벡터에 수직인 벡터  
3D : 어떤 평면에 수직인 벡터  
벅선벡트는 평면을 정의하는 데도 사용 할 수 있다. 점P와 P를 시점으로 하는 법선벡터 N이 있을 때 P를 포함 하는 평면의 표면이 어느 쪽을 향하는지 결정 할 수 있다.  

벡터는 기하학적인 벡터 공간을 구성하는 연산뿐 아니라 다른 종류의 연산도 부가적으로 정의 할 수 있다 - 내적, 외적  

- 내적 : 두 개의 벡터를 하나의 스칼라양으로 변환하는 연산
